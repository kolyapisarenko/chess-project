# Chess Project X ♟️

**Chess Project X** is a custom chess engine and gaming platform developed in Python using the Pygame library. The project combines classic board game mechanics with modern machine learning methods.

---

## 🚀 Main Concept

The project is divided into two strategic phases:

### Phase 1: Logic & UI Core
* **Proprietary Engine:** Development of a chess engine from scratch (no third-party logic libraries).
* **Complete Rulebook:** Implementation of castling, *en passant*, and pawn promotion.
* **Advanced UI:** Support for pre-moves, visual analysis (arrows), and dynamic move highlighting.

### Phase 2: Self-Learning AI
* **Neural Network Integration:** Implementation of artificial intelligence for move evaluation.
* **Evolutionary Mechanics:** The AI analyzes every played game, adapting its strategy and improving position assessment over time.

---

## 🎮 Game Modes

| Mode | Description |
| :--- | :--- |
| **PvP (Local)** | Traditional match for two players on a single machine. |
| **PvE (Adaptive AI)** | Challenge a neural network that evolves and grows stronger with every game. |

## 🎨 Assets

### Visuals
The visual components of this project utilize **JohnPablok's Improved Cburnett Chess Set**.
* **Source:** OpenGameArt.org
* **Author:** JohnPablok
* **Features:** Proportional scaling (HD 1x/2x sets), refined silhouettes (Knight/King), and custom wood-style squares.

### Audio
Sound effects (moves, captures, checks, and game-over states) are sourced from **Chess.com**.
* **Authenticity:** Provides a familiar and professional auditory experience for players.
* **Implementation:** Smart audio-triggering system to ensure zero-latency feedback and proper game-state synchronization.

### Key Features:
* **Proportional Scaling:** Utilizing 1x/2x PNG sets where pieces have natural height variations (e.g., pawns are shorter than rooks).
* **HD Quality:** Consistent line widths and refined silhouettes (specifically the modified Knight and King).
* **Custom Squares:** Brown/gray wood-style textures for a premium board feel.
